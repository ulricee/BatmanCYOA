## Building the game

1. Tested with Node.js version 24.14.1. Download Node.js from [nodejs.org](https://nodejs.org/), use [nvm](https://github.com/nvm-sh/nvm) for Unix systems, or [nvm-windows](https://github.com/coreybutler/nvm-windows) for Windows to manage Node.js versions.

**Note:** Make sure you are in the project directory before running the following commands. For example: `cd BatmanCYOA`

2. Use `npm install --upgrade https://github.com/aucchen/dendrynexus` to upgrade to the latest dendrynexus and install other dependencies.

3. Run `npx dendrynexus make-html` to build the game.

4. Open `out/html/index.html` to run the game.
